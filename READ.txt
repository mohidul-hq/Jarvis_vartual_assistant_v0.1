Jarvis - Python Virtual Assistant 🚀

📚 About the Project
Jarvis is a Python-based virtual assistant that automates everyday tasks using voice commands, helping you stay productive.

🎯 Key Features
🔊 Voice-Activated Commands: Talk to Jarvis for seamless task automation.
📧 Email Automation: Send emails using voice commands.
📱 WhatsApp Integration: Send messages and make voice/video calls.
🌐 Web Navigation: Open YouTube, Google, and other websites.
⏰ Time Queries: Get the current system time.
🎵 Music Control: Play YouTube playlists.
📖 Wikipedia Integration: Get instant knowledge summaries.
🛠️ Tech Stack
Programming Language: Python
Libraries Used:
pyttsx3, speech_recognition, wikipedia, pyautogui, psutil, smtplib, csv
